# Teensy Test Program (Teensy テストプログラムについて)

[English](#english) / [日本語](#日本語)

---

## English

### Overview
This folder contains the `teensy.ino` sketch along with several accompanying drum sound sample files. 

### Important Notice
* **Purpose:** This program is intended strictly for **testing specific hardware components**. It does not provide a full or comprehensive synthesizer experience.
* **Status:** This code and the related hardware designs are primarily a **record of development progress**.

### Requirements for Testing
To test this program, you will need the following hardware:
* Teensy 4.1
* Teensy Audio Board (Rev D)
* Suitable custom hardware based on our [KiCad Data](../ハードウェア/) (Not recommended for production)

> [!CAUTION]
> The current KiCad data is **incomplete**. We strongly advise **against** manufacturing a PCB or assembling a board using these files at this stage.

## 日本語

### 概要
このフォルダには、`teensy.ino` スケッチと、それに付属するドラムサウンドのサンプルファイルが格納されています。

### 注意事項
* **目的:** このプログラムは**ハードウェアの一部をテストするため**のものです。本格的なシンセサイザーとしての機能や体験を提供するものではありません。
* **位置づけ:** 本プログラムおよび関連データは、単なる**開発の進捗記録**のようなものです。

### テストに必要な環境
このプログラムを実際に動かしてテストするには、以下のハードウェアが必要です。
* Teensy 4.1
* Teensy Audio Board (Rev D)
* 適切なハードウェア（基板データは [KiCadデータ](../ハードウェア/) を参照）

> [!CAUTION]
> 現在公開しているKiCadデータ自体は**不完全なもの**です。そのため、このデータをもとに基板を発注・作成し、組み立てて動かすといったことは**推奨しません**。
